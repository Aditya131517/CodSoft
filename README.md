# CodSoft

# Python GUI Projects Collection

This repository contains three beginner-friendly Python GUI projects built using Tkinter. These projects focus on practical applications and help strengthen Python fundamentals and GUI development skills.

# Projects Included

### 1. To-Do List Application
A GUI-based task manager that helps users organize and track daily tasks.

# Features
- Add tasks
- Update tasks
- Delete tasks
- Mark tasks as completed
- Save tasks automatically using `tasks.txt`
- Load saved tasks when the app restarts

#Technologies Used
- Python
- Tkinter
- File Handling

---

# 2. Simple Calculator
A calculator with a graphical interface that performs basic arithmetic operations.

# Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface
- Error handling for invalid input and division by zero

# Technologies Used
- Python
- Tkinter

# 3. Password Generator
A password generator that creates secure and random passwords based on user preferences.

# Features
- Custom password length
- Lowercase letters option
- Uppercase letters option
- Numbers option
- Symbols option
- Generates strong random passwords

# Technologies Used
- Python
- Tkinter
- Random module
- String module
